### 2. 修复的关键功能

#### ✅ 严格模式 (strict_mode)
- 控制回答只基于知识库内容
- 不添加推测或假设
- 信息不完整时明确说明

#### ✅ 通用知识限制 (allow_general_knowledge)
- 控制是否允许使用 LLM 训练数据中的知识
- 严格限制只使用知识库内容
- 或允许适当补充通用知识

#### ✅ 引用要求 (require_citations)
- 强制要求回答包含引用标记
- 使用 [1]、[2] 等数字标记
- 每个关键信息都有对应引用

#### ✅ 禁止主题 (forbidden_topics)
- 在 Prompt 中明确禁止的主题
- 例如：薪资、工资等敏感话题
- LLM 会拒绝回答相关问题

#### ✅ 禁止关键词 (forbidden_keywords)
- 在 Prompt 中明确禁止的关键词
- 回答中不应包含这些词汇
- 例如：工资、薪水等

#### ✅ 答案长度限制 (max_answer_length)
- 限制回答的最大字符数
- 超过限制自动截断
- 记录警告日志

#### ✅ 答案验证集成
- 在生成回答后自动验证
- 检查置信度、来源归属、幻觉检测
- 低置信度回答附加警告信息
